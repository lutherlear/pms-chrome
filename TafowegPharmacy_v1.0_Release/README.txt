FIRST TIME SETUP
================================================================================

1. Launch the application
2. Login with admin credentials (admin/admin123)
3. Go to Settings > Company to configure your pharmacy details
4. Navigate to Users to change default passwords
5. Add your drug inventory in the Inventory section
6. Configure receipt settings as needed
7. Create additional user accounts for your staff

================================================================================
DATABASE LOCATION
================================================================================

The application database is stored at:
%APPDATA%\databases\pharmacy.db

It is recommended to regularly backup your database using the 
Settings > Backup & Restore feature.

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Application won't start
Solution: 
- Ensure Windows 10 or later is installed
- Check if Microsoft Visual C++ Redistributable is installed
- Run as Administrator if needed

Issue: "Database error" message
Solution:
- Check disk space availability
- Ensure the app has write permissions
- Try running as Administrator

Issue: Reports not generating
Solution:
- Check if data exists for the selected period
- Clear cache and restart application
